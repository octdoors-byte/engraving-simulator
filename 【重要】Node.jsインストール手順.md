# Node.js インストール手順

## 現在の状況

`統合用ビルド実行.bat` を実行したところ、`pnpm` も `npm` も見つかりませんでした。

これは、**Node.js がインストールされていない**か、**PATH に含まれていない**可能性があります。

---

## 解決方法

### ステップ1: Node.js のインストール確認

1. **コマンドプロンプトを開く**
   - Windowsキー + R → `cmd` と入力 → Enter

2. **以下のコマンドを実行:**
   ```cmd
   node --version
   ```

3. **バージョンが表示されれば、Node.js はインストール済みです**
   - 次のステップへ進んでください

4. **エラーが出る場合は、Node.js をインストールしてください**

---

### ステップ2: Node.js のインストール（必要な場合）

1. **Node.js の公式サイトにアクセス**
   - https://nodejs.org/

2. **LTS版（推奨版）をダウンロード**
   - 「推奨版」と表示されているバージョンをダウンロード

3. **インストーラーを実行**
   - ダウンロードした `.msi` ファイルをダブルクリック
   - インストールウィザードに従ってインストール
   - **「Add to PATH」にチェックが入っていることを確認**

4. **コンピューターを再起動**
   - インストール後、必ず再起動してください

5. **インストール確認**
   - コマンドプロンプトを開く
   - 以下のコマンドを実行:
     ```cmd
     node --version
     npm --version
     ```
   - 両方ともバージョンが表示されればOK

---

### ステップ3: ビルドの実行

Node.js がインストールされたら、以下のいずれかの方法でビルドを実行してください:

**方法1: 修正版バッチファイル**
```
統合用ビルド実行.bat
```

**方法2: 手動確認バッチファイル**
```
統合用ビルド実行_手動確認.bat
```

**方法3: コマンドプロンプトで直接実行**
```cmd
npm run build:integrated
```

---

## トラブルシューティング

### Node.js をインストールしたのに、まだエラーが出る場合

1. **コンピューターを再起動**
   - PATH の変更を反映するために必要です

2. **コマンドプロンプトを新しく開く**
   - 既に開いているコマンドプロンプトは閉じて、新しく開いてください

3. **環境変数の確認**
   - Windowsキー + Pause → 「システムの詳細設定」
   - 「環境変数」をクリック
   - 「システム環境変数」の「Path」を確認
   - Node.js のパスが含まれているか確認

---

## 注意事項

- Node.js をインストールすると、`npm` も自動的にインストールされます
- `pnpm` は別途インストールが必要ですが、`npm` があればビルドは可能です
- ビルドには時間がかかることがあります（1〜3分程度）

---

## 次のステップ

Node.js のインストールが完了したら:

1. `統合用ビルド実行.bat` を実行
2. `dist/index.html` のパスが `/simulator/assets/` になっているか確認
3. FTPでアップロード
