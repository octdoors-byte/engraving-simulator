# 仕様メモ（カテゴリ一覧 & カラー対応）

## 追加機能
- `/categories` ページ  
  - カテゴリごとにテーブルを分割表示。  
  - 行構成: テンプレート名 / 公開URL / 操作（コピー・開く）。  
  - 対象: 公開中テンプレートのみ。front/back がある場合は表面を優先して URL を生成。  
  - URL には `?cat=カテゴリID` を付与してカテゴリ絞り込み状態を維持。

- 公開テンプレート一覧（`/top`）  
  - カテゴリバッジを頭文字1文字で表示。  
  - カテゴリカラーを反映（未指定時は従来グレー）。

- カテゴリカラー設定  
  - 基本設定 > カテゴリ編集で 5 色プリセットから選択。  
  - 新規カテゴリ追加時にプリセット色を自動割当。  
  - `commonInfoCategories` に `color` を保存。

## データ構造
- `CommonSettings.commonInfoCategories`: `{ id, title, body?, color? }`
  - `color` 省略時は既定グレーで表示。

## 注意点
- カラーは固定 5 色（`CATEGORY_COLORS`）。変更は配列編集で対応。  
- 既存データは `color` 未指定のままでも動作する。  
- 管理側一覧への色表示は未対応（必要ならバッジに同色を適用する）。***
