テンプレート作成ガイド

template.json と背景画像を用意して、公開できるテンプレートを作る手順をまとめました。

使い方のポイント

  • 必要ファイルは「template.json」と背景画像の2つ。
  • 背景サイズと engravingArea の座標を一致させる。
  • 表裏を分けたいときはキーを xxx_front / xxx_back にする。

基本の流れ

  1. 背景画像を実寸で作成し、template.json に同じサイズを設定。
  2. engravingArea や placementRules を用途に合わせて記入。
  3. テンプレート管理画面で JSON + 背景を同時にドロップして登録。
  4. 公開したい場合は状態を published に変更。

アイコン・ボタンの意味

  • ⬆️ ドロップエリア: JSON と背景を同時に登録。
  • 🖼 プレビュー: 背景と座標が合っているか確認。
  • ✅ 状態: draft/tested/published/archive を切替。

よくある質問

  • 座標がズレる: 背景サイズと JSON の canvasWidth/Height が一致しているか再確認。
  • 登録エラーが出る: キー重複やファイル名不一致がないか確認。
  • 表裏を1行にまとめたい: xxx_front/xxx_back を同じベース名にすると公開一覧でまとまります。
