テンプレート管理ガイド

テンプレ登録・状態変更・カテゴリ設定の基本操作を、専門用語なしでまとめました。

使い方のポイント

  • 登録は「template.json + 背景画像」を同時にドロップ。
  • 状態は draft → tested → published → archive。公開一覧に出るのは published だけ。
  • カテゴリは複数選択可。公開バッジに色が反映されます。

基本の流れ

  1. 新規登録で template.json と背景画像をアップロード。
  2. 表示名・カテゴリを編集（ダブルクリック）。
  3. 状態を published にして公開。不要なら archive で非表示。

アイコン・ボタンの意味

  • ⬆️ ドロップエリア: JSON と背景を同時に登録。
  • ✅ 状態: draft/tested/published/archive を切替。
  • 🖼 プレビュー: 背景画像を確認。
  • 🗑 削除: デザインが紐づいていると削除不可（先にarchive推奨）。

よくある質問

  • 登録できない: JSON と背景のファイル名が一致しているか、キーが重複していないか確認。
  • サイズがずれる: 背景と engravingArea のサイズ差は自動補正されますが、プレビューで必ず確認。
  • 公開一覧に出ない: 状態が published か、カテゴリ未設定でフィルタされていないか確認。
