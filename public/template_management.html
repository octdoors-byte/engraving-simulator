操作手順

「合成レザー専用盾窓」などの新しい商品テンプレートはここから追加します。

ℹ️
同期について
登録するJSONファイルと背景画像は、システム内部で完全同期して保存されます。必ずセットで扱ってください。

新規登録の手順（5ステップ）

  1. ファイルを準備する 情報付きの template.json と、対応する背景画像を用意します。
    ※ 画像のファイル名は JSON に書いた名前と同じである必要があります
  2. ドラッグ&ドロップで登録 画面上部の「新規登録」エリアに、2つのファイルをまとめてドラッグ&ドロップします。
    ※表示：『ここに template.json と背景画像をまとめて置いてください』
    💡 2つのファイルを同時に選択してドラッグ&ドロップするか、ファイル選択ダイアログで両方を選択してください。
  3. 状態を選択する 一覧に追加された行のステータスをプルダウンから選択します。
    （選択肢：下書き テスト済み 公開中 アーカイブ）
  4. 表示を確認する テスト スマホ表示 PC表示 ボタンを押し、シミュレーターPDFなどで正しく表示されるか直接確認します。
    ※ 「テスト」ボタンは下書き状態の場合、自動的に「テスト済み」に変更します
  5. 表示名・カテゴリを調整する 一覧の「表示名」をダブルクリックすると名称とカテゴリの変更が可能です。お客様にわかりやすい名前に修正してください。
    ※ Enterキーで保存、Escapeキーでキャンセル
    💡 表示名はお客様に表示される名前です。商品名や用途が分かる名前にすると便利です。

状態遷移のルール

テンプレートの状態は以下のように遷移できます。一度「アーカイブ」にすると、編集や状態変更ができなくなります。

下書き → テスト済み → 公開中 → アーカイブ

各状態の説明

  • 下書き：登録直後の状態。テストや編集が可能です。
  • テスト済み：動作確認が完了した状態。公開前にテスト済みに変更します。
  • 公開中：お客様に表示される状態。「公開テンプレート一覧」に表示されます。
  • アーカイブ：非公開状態。編集・状態変更・削除ができません。復帰はできません。
⚠️
アーカイブについて
一度「アーカイブ」にすると、編集・状態変更・削除が一切できなくなります。また、アーカイブから他の状態に戻すことはできません。アーカイブにする前に必ず内容を確認してください。

主な操作ボタン

⚠️ 注意点とよくあるミス

⚠️
「保存データを初期化」ボタンに注意
画面内にある「保存データを初期化」を押すと、登録済みのテンプレートが全て消去される可能性があります。メンテナンス時以外は押さないようにしてください。

よくあるミス

  • ファイルが1つだけ登録できない
    template.json と背景画像は必ず2つまとめてドラッグ&ドロップしてください。片方だけでは正しく登録されません。
  • 画像のファイル名が一致しない
    template.json の background.fileName に記載されたファイル名と、実際の画像ファイル名が一致している必要があります。
  • お客様に表示されない
    「状態」が 下書き や アーカイブ になっていませんか？
    公開する場合は必ず 公開中 に変更してください。
  • 削除できない
    デザインが紐づいているテンプレートは削除できません。エラーメッセージが表示されます。
    削除したい場合は、まず「アーカイブ」に変更してください。
  • 同じキーが既に登録されている
    template.json の templateKey が既に登録されている場合、エラーになります。
    別のキー名を使用するか、既存のテンプレートを削除/アーカイブしてください。

エラーメッセージの対処法

「template.json と背景画像を同時に選択してください」
JSONファイルと画像ファイルの両方を選択してください。2つのファイルをまとめてドラッグ&ドロップするか、ファイル選択ダイアログで両方を選択してください。
「背景画像のファイル名が template.json と一致しません」
template.json の background.fileName に記載されたファイル名と、実際の画像ファイル名が完全に一致している必要があります。大文字小文字も区別されます。
「canvasWidth/Height が範囲外です」
template.json の canvasWidth と canvasHeight が、背景画像のサイズと一致している必要があります。画像の実際のサイズを確認してください。
「刻印枠がキャンバス外です」
template.json の刻印枠（stampAreas）の座標が、キャンバスサイズを超えています。座標を修正してください。
「同じキーが既に登録されています」
template.json の templateKey が既に登録されています。別のキー名を使用するか、既存のテンプレートを削除/アーカイブしてください。
「デザインが紐づいているため削除できません。アーカイブにしてください」
このテンプレートを使用して作成されたデザインが存在するため、削除できません。非公開にしたい場合は「アーカイブ」に変更してください。
「アーカイブは編集できません」
アーカイブ状態のテンプレートは編集・状態変更・削除ができません。編集が必要な場合は、新しいテンプレートとして登録してください。

よくある質問

表示名とカテゴリはどこで編集できますか？
一覧テーブルの「表示名」列をダブルクリックすると、表示名とカテゴリを編集できます。Enterキーで保存、Escapeキーでキャンセルです。
モノクロ設定とは何ですか？
ロゴをモノクロ（白黒）表示にする設定です。チェックボックスで有効/無効を切り替えられます。アーカイブ状態では変更できません。
テスト、スマホ表示、PC表示の違いは？
「テスト」は下書き状態の場合、自動的に「テスト済み」に変更してからシミュレーターを開きます（スマホサイズ）。「スマホ表示」と「PC表示」は、それぞれスマホサイズ（390×844）とPCサイズ（1280×720）でシミュレーターを開きます。
背景画像のサイズが合わない場合は？
システムが自動的に背景画像のサイズに合わせてテンプレートを補正します。補正が行われた場合は、通知メッセージが表示されます。
テンプレートを削除したら復元できますか？
削除したテンプレートは復元できません。削除前に必ず内容を確認してください。非公開にしたいだけの場合は「アーカイブ」に変更してください。
