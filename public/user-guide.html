公開URLを開いて、ロゴを置いて、PDFを送るだけ。初めてでもすぐ終わります。

3ステップで完了 初めてでも簡単

大まかな流れは「URLを開く → ロゴを置く → PDFを送る」の3つだけ。画面のイメージとセットで確認できます。

1
まずはURLを開く

ショップから届いた公開URLをクリックするだけで、対象商品の説明ページが開きます。

商品名と説明をざっと読んで、チェックを入れて「編集を開始する」を押します。

2
ロゴを置く

「画像をアップロード」からロゴを選択（2MB以内推奨）。背景が透明ならそのままOK、余白はトリミングでカット。

ガイド枠からはみ出さないように、位置と大きさだけ合わせれば完了です。

3
見た目を確認してPDF

プレビューで仕上がりを確認し、「PDFを発行」を押します。PDFをショップに送れば手続き完了です。

PDFに記載のデザインIDも合わせて伝えると、対応が早くなります。

操作のポイント

推奨環境: 最新 Edge / Chrome / Firefox / Safari デザイン保存: ブラウザに一時保存 ロゴ画像: 2MB 以内推奨
  • 公開URLは商品専用です。届いたものだけ開いてください。
  • 透明PNGがおすすめ。余白はトリミングでカットして枠内に収めます。
  • 途中のデザインはブラウザに一時保存。別端末ではやり直しになります。
  • 重いときは他のタブを閉じるかブラウザを再起動してください。
✔ ロゴは透明PNGがおすすめ（白背景だと枠が出ることあり）
✔ 余白はトリミングでカットしてから配置
✔ 印刷枠からはみ出していないかプレビューで確認
✔ PDFのデザインIDをショップへ伝えると対応が早いです
