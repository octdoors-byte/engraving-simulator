名入れ刻印シミュレーター ご利用ガイド

公開URLを開いて、ロゴを置いて、PDFを送るだけ。初めてでもすぐ終わります。

基本の流れ お客様操作のみ

画面イメージ

最初に共通説明→ロゴ配置→PDF発行の順に進みます。イメージは下のサンプルをご覧ください。

1
テンプレートを開く
  • ショップから届いた「公開URL」を開くだけ。
  • 表示されたテンプレートが商品と合っているかだけ確認します。
2
共通説明を読む
  • 最初に注意事項が出ます。ざっと読んでチェックを入れます。
  • 「編集を開始する」でシミュレーターへ進みます。
3
ロゴをアップロード
  • 「画像をアップロード」からロゴを選ぶだけ（2MB 以内推奨）。
  • 背景が透明ならそのままOK。余白はトリミングでカットできます。
  • 印刷枠からはみ出さないよう位置と大きさを合わせます。
4
仕上がりを確認
  • プレビューで見た目を確認し、「PDFを発行」します。
  • PDFをショップへ送れば注文は完了です。

操作のポイント

推奨環境: 最新 Edge / Chrome / Firefox / Safari デザイン保存: ブラウザに一時保存 ロゴ画像: 2MB 以内推奨
  • 公開URLは商品専用です。届いたものだけ開いてください。
  • ロゴは印刷枠に収まるように。透明PNGだと仕上がりがきれいです。
  • 余白が気になればトリミングで切り取ってから配置すると楽です。
  • 途中のデザインはブラウザに一時保存されます。別端末ではやり直しになります。
  • 動作が重いときは他のタブを閉じるかブラウザを再起動してください。
✔ ロゴは透明PNGがおすすめ（白背景だと枠が出ることあり）
✔ 余白はトリミングでカットしてから配置
✔ 印刷枠からはみ出していないかプレビューで確認
✔ PDFのデザインIDをショップへ伝えると対応が早いです

お問い合わせ

PDFに記載されるデザインIDを合わせてお伝えいただくと、対応がスムーズです。
