テンプレートを選び、ロゴを配置して、確認用PDFを発行するまでの手順をまとめました。

基本の流れ

1. テンプレートを選ぶ
  • 公開テンプレート一覧（トップページ）で商品に合うテンプレートを選びます。
  • 「公開URL」を開くとシミュレーターが起動します。
2. 共通説明を確認する
  • 「共通説明＋公開URL」から開くと、対象商品の注意事項をまとめて確認できます。
  • 内容を読んでチェックを入れ、「編集を開始する」でシミュレーターへ進みます。
3. ロゴをアップロードする
  • 「画像をアップロード」ボタンからロゴ画像を選択します。
  • 配置範囲（刻印エリア）に合わせて位置とサイズを調整します。
4. 配置を確認する
  • プレビューで仕上がりを確認し、問題なければ「PDFを発行」します。
  • PDFは刻印イメージの確認用です。内容をショップへ共有してください。

操作のポイント

推奨環境: 最新 Edge / Chrome / Firefox / Safari データ保存: ブラウザに保存されます 画像上限: 2MB 以内推奨
  • テンプレートは商品ごとに公開されています。誤ったテンプレートを使わないようご注意ください。
  • ロゴの上下左右はガイド線（刻印範囲）からはみ出さないように配置してください。
  • 作業途中のデザインはブラウザに保存されます。別端末では再度デザインが必要です。
  • 動作が重い場合は、他のタブを閉じるかブラウザを再起動してお試しください。

お問い合わせ

操作に不明点がある場合や、刻印内容の個別相談が必要な場合はショップまでご連絡ください。

PDFに記載されるデザインIDを合わせてお伝えいただくと、対応がスムーズです。
