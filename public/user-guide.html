テンプレートを選び、ロゴを配置して、確認用PDFを発行するまでの手順をまとめました。

基本の流れ

1. テンプレートを選ぶ
  • ショップが案内する「公開URL」を開き、対象商品のテンプレートを表示します。
  • 商品と違うテンプレートを開かないよう、URLを再確認してください。
2. 共通説明を確認する
  • 先に共通説明ページが表示されます。対象商品の注意事項を必ずお読みください。
  • 内容を読んでチェックを入れ、「編集を開始する」でシミュレーターへ進みます。
3. ロゴをアップロードする
  • 「画像をアップロード」ボタンからロゴ画像を選択します（2MB 以内推奨）。
  • 刻印エリアの枠からはみ出さないように位置とサイズを調整します。
4. 配置を確認する
  • プレビューで仕上がりを確認し、問題なければ「PDFを発行」します。
  • PDFは刻印イメージの確認用です。ショップへ送付してご注文を進めてください。

操作のポイント

推奨環境: 最新 Edge / Chrome / Firefox / Safari デザイン保存: ブラウザに一時保存 ロゴ画像: 2MB 以内推奨
  • テンプレートは商品専用です。案内された公開URL以外は使わないでください。
  • ロゴは刻印エリアからはみ出さないように調整してください。
  • 作業途中のデザインはブラウザに一時保存されます。別端末では再度デザインが必要です。
  • 動作が重い場合は、他のタブを閉じるかブラウザを再起動してお試しください。

お問い合わせ

操作に不明点がある場合や、刻印内容の個別相談が必要な場合はショップまでご連絡ください。

PDFに記載されるデザインIDを合わせてお伝えいただくと、対応がスムーズです。
