基本設定（お客様向け共通説明ページ）操作ガイド

シミュレーターのトップ画面にお客様向けに表示される情報を設定・編集します。
対象：システム管理者・サイト運営担当者

シミュレーターのトップ画面にお客様向けに表示される情報を設定・編集します。

📝

1. 説明文を設定

タイトルや説明文、よくある質問を入力します。

🖼️

2. 画像を追加

説明用の画像をアップロードできます（各2MBまで）。

👁️

3. プレビューで確認

保存前にプレビューで表示を確認してから保存します。

画面イメージと構成

設定・編集の手順

① ヘッダー/フッター設定

画面上部・下部の共通エリアの設定です。以下の項目を設定できます：

  • ロゴ画像：ファイルを選択すると自動保存されます（JPG/PNG推奨）
  • トップのタイトル：シミュレーターのメインタイトル（例：「刻印印刷シミュレーター」）
  • ヘッダーテキスト：画面上部に表示するテキスト
  • フッターテキスト：画面下部に表示するテキスト（例：「このシミュレーションはイメージです」）
  • ヘッダー配置：左 / 中央 / 右から選択
  • フッター配置：左 / 中央 / 右から選択
  • ロゴサイズ：小 / 中 / 大から選択
  • ヘッダー文字サイズ：小 / 中 / 大から選択
  • フッター文字サイズ：小 / 中 / 大から選択

💡 ポイント：ロゴ画像は選択すると自動保存されますが、その他の設定は「保存する」ボタンで反映してください。

② タイトル / フッター文言

シミュレーターのメインタイトル（例：「刻印印刷シミュレーター」）や、フッターに表示する注釈（例：「このシミュレーションはイメージです」等）を入力します。

③ 共通説明テキスト

お客様が最初に目にする「ご利用前のご案内」を入力します。利用の流れや注意点などを記載してください。

④ よくある質問（FAQ）の追加・編集

FAQはカード形式で表示されます。テキストエリアに直接入力するか、「ひな型をコピー」ボタンでテンプレートをコピーして使用できます。

📝 入力ルール（重要！）：

  • 質問行：「Q.」で始める（例：「Q. 推奨ブラウザは？」）
  • 回答行：質問の次の行から続けて書く（空行を入れない）
  • 複数行の回答：改行を続けて書く
  • 次のカード：空行1つを入れて区切る

入力例：

Q. 推奨ブラウザは？
最新の Edge / Chrome / Firefox / Safari でご利用ください。

Q. 作ったデザインは保存できますか？
デザインIDを発行すると保存されます。デザインIDを控えておけば、同じデザインで再製作できます。

💡 コツ：「ひな型をコピー」ボタンを押すと、サンプルのFAQがクリップボードにコピーされます。それを貼り付けて編集すると効率的です。

⑤ レイアウト / 画像添付

レイアウト設定：画像とテキストの配置を選択します。

  • 画像を上 / テキストを下
  • テキストを上 / 画像を下
  • 画像が左 / テキストが右
  • テキストが左 / 画像が右

画像添付：説明用の補足画像をアップロードできます。

  • 「画像を選択」ボタンをクリックしてファイルを選択
  • ※ 画像サイズ上限：各2MBまで、最大5枚
  • 画像の並び替え：画像をドラッグ&ドロップするか、各画像の「↑」「↓」ボタンで順序を変更
  • 画像の削除：各画像の「削除」ボタンをクリック

💡 コツ：画像はドラッグ&ドロップで簡単に並び替えられます。画像の上に「⇅ 並び替え」と表示されている部分をドラッグしてください。

⑥ 補足資料（PDF）の添付

補足資料としてPDFファイルを添付できます。
※ PDFサイズ上限：5MBまで
お客様がダウンロードできる形式で表示されます。

⑦ プレビューで確認 → 保存

プレビュー確認：

  • 画面下部のプレビューエリアで、実際のお客様からの見え方を確認できます
  • 「プレビューを更新」ボタンで最新の設定を反映
  • 「新しいタブで開く」ボタンで別タブで確認も可能

保存：

  • 問題なければ「保存する」ボタンを押して反映させます
  • ※ 変更がある場合のみ「保存する」ボタンが有効（青色）になります
  • 変更がない場合は「保存する」ボタンは無効（グレーアウト）のままです
  • 保存すると即時に本番反映されます

⚠️ 重要：保存する前に必ずプレビューで確認してください。保存すると即時に本番反映されます。

⚠️ 運用上の注意点

  • 保存ボタンの確認：変更を加えると「保存する」ボタンが有効になります。必ず「保存する」ボタンを押して設定を反映してください。
  • 公開前のプレビュー必須：保存すると即時に本番反映されます。必ずプレビューで崩れがないか確認してください。
  • 文言はお客様に見えます：社内用語ではなく、お客様に伝わる言葉で入力してください。
  • 画像サイズ制限：各画像2MBを超える画像はアップロードできません。事前にリサイズや圧縮を行ってください。最大5枚まで登録できます。
  • PDFサイズ制限：5MBを超えるPDFはアップロードできません。必要に応じて圧縮してください。
  • 変更の影響範囲：ここでの設定は、すべてのテンプレート共通の「導入ページ」に影響します。
  • 自動保存について：一部の項目（ロゴ画像など）は自動保存されますが、基本的には「保存する」ボタンで手動保存してください。

管理者向けFAQ

設定した内容はどこに表示されますか？
シミュレーターのURLを開いた直後の「トップページ（導入画面）」、または「共通説明 + 公開URL」をクリックした際に表示される共通説明ページ（/common）に表示されます。
保存した内容はいつ反映されますか？
「保存する」ボタンを押すと即時に反映されます。ブラウザのキャッシュにより古い情報が残る場合があるため、確認時はスーパーリロード（Ctrl+F5等）を推奨します。
アップロードした画像が表示されません
画像形式（JPG/PNG推奨）やサイズ（各2MB以下、最大5枚）を確認してください。また、特殊な文字コードを含むファイル名は避けてください。
画像の順序を変更したい
画像一覧で画像をドラッグ&ドロップすることで、順序を変更できます。変更後は「保存する」ボタンを押して反映してください。
「保存する」ボタンが無効になっています
変更がない場合、「保存する」ボタンは無効（グレーアウト）になります。設定を変更すると自動的に有効（青色）になります。
💡 ロゴ画像は自動保存されますが、その他の設定は「保存する」ボタンで手動保存が必要です。
FAQの入力方法がわかりません
FAQは特定のフォーマットで入力します：
  • 質問は「Q.」で始める
  • 回答は質問の次の行から続けて書く（空行を入れない）
  • 複数のカードは空行1つで区切る
「ひな型をコピー」ボタンでサンプルをコピーして、それを参考に編集すると分かりやすいです。
画像の順序を変更する方法は？
画像の順序は以下の方法で変更できます：
  • ドラッグ&ドロップ：画像をドラッグして別の位置にドロップ
  • ↑↓ボタン：各画像の「↑」「↓」ボタンをクリック
変更後は「保存する」ボタンを押して反映してください。

カテゴリ設定（カラー対応）

公開テンプレートをカテゴリで整理し、バッジカラーを設定できます（最大3件、5色＋黒から選択）。

  • IDは自動採番。タイトル・説明は自由入力（未入力の場合は「未設定」と表示）。
  • カラーはプリセットからクリック選択。公開一覧のバッジ色にも反映されます。
  • 追加時は自動で次のプリセット色が割り当てられます。不要になったカテゴリは削除してください。
  • 公開中ステータスのテンプレのみカテゴリ一覧（/categories）に表示されます。

※ front/back のあるテンプレは表面が優先され、URLに ?cat=カテゴリID が付きます。
