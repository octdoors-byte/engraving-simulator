基本設定ガイド

ロゴ・ヘッダー/フッター文言・共通説明・カテゴリ設定など、初期設定のポイントをまとめました。

基本設定でできること

  • ロゴ画像の設定、表示位置・サイズ（S/M/L）、ヘッダー/フッター文言と揃え位置。
  • 共通説明（タイトル・本文・画像・PDF・FAQ）の登録とレイアウト選択。
  • カテゴリの作成（最大3件）とカラー指定（5色＋黒）。

画像・PDFアップロード

  • 共通説明の画像: 最大 5 枚、1 枚あたり 2MB まで。
  • 共通説明の PDF: 1 ファイル、5MB まで。
  • アップ後は「保存する」ボタンで反映してください。

カテゴリ設定（カラー対応）

  • 最大 3 件まで追加可能。タイトル・説明は任意。未入力の場合は「未設定」と表示。
  • カラーはプリセット（5色＋黒）からクリック選択。公開テンプレ一覧のバッジにも反映。
  • 追加時は次のプリセット色を自動割当。不要なカテゴリは削除してください。
  • 公開中ステータスのテンプレだけがカテゴリ一覧（/categories）に表示されます。
  • front/back があるテンプレは表面を優先し、URL には ?cat=カテゴリID が付きます。

バックアップと復元

  • 「バックアップを保存」で設定・テンプレ・デザインを JSON 出力（localStorage/IndexedDB が対象）。
  • 「バックアップを復元」で JSON から上書き復元（現在のデータは置き換わります）。
  • 「保存データを初期化」は全削除するため、事前にバックアップ推奨。

保存のコツ

  • 入力後は必ず「保存する」ボタンを押して確定。
  • 画像・PDFアップ後も保存ボタンで反映されます。
  • カテゴリを変更したら、公開テンプレ一覧・カテゴリ一覧で色と表示を確認してください。
